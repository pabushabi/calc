# CALC

Simple calculator on C#

#### Now working these functions:

* `+`
* `-`
* `*`
* `/`
* `,`
* `%`
* `!`
* `^`
* `sin`
* `cos`
* `tan` 
* `ln`
* `e`
* keyboard input (not all operations):
	* `0-9` - digital keys
	* `*` - Shift + 8
	* `^` - Shift + 6
	* `%` - Shift + 5
	* `(` - Shift + 9
	* `)` - Shift + 0
	* `+` - Shift + A
	* `-` - Shift + S
	* `/` - Shift + D
	* `cos` - C
	* `sin` - S
	* `tan` - T
	* `ln` - L
	* `sqrt` - Q
	* extra - X
	* copy result to clipboard - Ctrl + C
	* paste from clipboard - Ctrl + V
	* calculate the result - Tab
	* erase - Backspace
* history
* auto closing brackets
